# Практические работы 13-14 — Offline TODO + PWA

Простое веб-приложение для управления задачами.

Что реализовано:
- просмотр списка задач
- добавление задач
- сохранение в `localStorage`
- Service Worker: регистрация и кэширование статических ресурсов
- офлайн-режим: страница и интерфейс работают без сети
- Web App Manifest + иконки (минимум 3 PNG)

## Запуск

Service Worker работает только по HTTP(S), поэтому нужно запускать через локальный сервер.

Вариант 1 (Python):
```bash
cd practice-13-14-offline-todo-pwa
python -m http.server 5174
```

Откройте:
- `http://localhost:5174/`

## Проверка офлайн

1. Откройте приложение один раз онлайн (чтобы SW закэшировал ресурсы).
2. В DevTools включите Offline (Network).
3. Перезагрузите страницу — она должна загрузиться из кэша.
4. Добавьте задачу — она сохраняется в `localStorage` и остается после перезагрузки.

## Установка (PWA)

1. Откройте приложение в Chrome.
2. Установите (иконка установки в адресной строке или меню).
3. Убедитесь, что приложение запускается в отдельном окне (display: standalone).
